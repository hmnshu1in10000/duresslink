How SecureNote Works

  • ■ Standard Links (Unshortened) 100% decentralized. All encrypted payloads live natively inside the URL itself. These links can never be blocked, traced, or deleted, and will work forever.
  • ■ Shortened Links The app securely stores the URL routing map in a developer-grade global registry (is.gd) for permanent lifespans. Shortened links retain your custom domain name and are resolved client-side in the background, keeping all security parameters intact.
